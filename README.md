# Requirements
iOS 8.0+

# Installation

### CocoaPods
To integrate ZHIconFontKit into your Xcode project using CocoaPods, specify it in your Podfile:

``` ruby
  use_frameworks!

  pod ‘ZHIconFontKit’

```
### Manual
Just copy the files which under IconFontKit folder to your project
